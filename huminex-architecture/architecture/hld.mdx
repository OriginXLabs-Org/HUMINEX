# High-Level Design

```mermaid
flowchart LR
  U[Users: Employer, Employee, Internal Admin] --> W[Frontend Web: React, Vite, MSAL]
  W --> I[NGINX Ingress: api.gethuminex.com]
  I --> A[HUMINEX API: .NET 8]

  A --> P[(Azure PostgreSQL)]
  A --> R[(Azure Cache for Redis)]
  A --> S[(Azure Service Bus)]
  A --> B[(Azure Blob Storage)]
  A --> K[Azure Key Vault]
  A --> C[Azure App Configuration]

  A --> O[OpenTelemetry and Serilog]
  O --> L[Log Analytics]
  O --> AP[Application Insights]

  L --> AMG[Azure Managed Grafana: Unified UI]
  AP --> AMG
  PROM[Managed Prometheus: Azure Monitor] --> AMG

  AP --> WB[HUMINEX Workbook]
  L --> AL[Azure Monitor Alerts]
  AL --> LA[Logic App Slack Relay]
```
