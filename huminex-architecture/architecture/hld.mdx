# High-Level Design

```mermaid
flowchart LR
  U[Users<br/>Employer, Employee, Internal Admin] --> W[Frontend Web<br/>React + Vite + MSAL]
  W --> I[NGINX Ingress<br/>api.gethuminex.com]
  I --> A[HUMINEX API<br/>.NET 8]

  A --> P[(Azure PostgreSQL)]
  A --> R[(Azure Cache for Redis)]
  A --> S[(Azure Service Bus)]
  A --> B[(Azure Blob Storage)]
  A --> K[Azure Key Vault]
  A --> C[Azure App Configuration]

  A --> O[OpenTelemetry + Serilog]
  O --> L[Log Analytics]
  O --> AP[Application Insights]

  L --> AMG[Azure Managed Grafana<br/>Unified UI]
  AP --> AMG
  PROM[Managed Prometheus<br/>(Azure Monitor)] --> AMG

  AP --> WB[HUMINEX Workbook]
  L --> AL[Azure Monitor Alerts]
  AL --> LA[Logic App Slack Relay]
```
