# CI/CD and GitOps Delivery

```mermaid
flowchart LR
  DEV[Developer Push -> main] --> GHA[GitHub Actions]
  GHA --> BCI[Backend CI]
  GHA --> FCI[Frontend CI]
  GHA --> E2E[E2E Playwright]
  GHA --> IMG[Backend Image Push -> ACR]

  IMG --> GITOPS[Update kustomization newTag]
  GITOPS --> ARGO[Argo CD<br/>huminex-platform-dev]
  ARGO --> AKS[AKS Rollout]
  AKS --> VERIFY[Health/Smoke Verification]

  GHA --> SLACK[Pipeline Status to Slack]
```
